### Auto-Refresh Settings
```
KPI Cards:          10 seconds (lightweight queries)

Tables (Live):      5 seconds (real-time feel)
Tables (Historical):60 seconds (heavy queries)
```

### Query Optimizations
```sql
-- Use latest window pattern (efficient)
WITH latest AS (
  SELECT MAX(window_start) as max_window
  FROM gold_table
)
SELECT g.* FROM gold_table g
INNER JOIN latest l ON g.window_start = l.max_window;

-- Always include LIMIT for unbounded queries
SELECT * FROM silver_table
ORDER BY timestamp DESC
LIMIT 100;  -- Prevents full table scan
```

### Caching Strategy
- Enable query result caching (1-5 minutes)
- Use dashboard-level cache for consistency
- Refresh cache on data updates
